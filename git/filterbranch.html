# github 특정 파일 모든 commit 기록에서 삭제하기

노출되면 안되는 정보가 이미 커밋을 해버려서 올라간 상황인데 해당 파일만 따로 제외 시키고 싶을때 어떻게 하나?

파일명이 .env인 경우에


// 모든 커밋기록에 해당 파일명을 찾아 제거
git filter-branch --tree-filter 'rm -rf .env' HEAD

후에


git push origin main
